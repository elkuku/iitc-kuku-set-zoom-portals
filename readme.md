# Set Zoom Portals

This is a plugin for [IITC](https://iitc.app).

It will add a button to zoom in or out to set the zoom level to "portals" (15).

## Installation

Visit the [release page](https://github.com/elkuku/iitc-kuku-set-zoom-portals/releases/latest) and install the `user.js` file
